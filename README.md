# ld7182
This project introduces a smart energy monitoring system based on artificial intelligence and various other techniques: cloud communication, environmental sensors, and machine learning techniques are developed with ESP32. The parameters that were monitored were temperature, humidity, light intensity, and motion detection all within a simulated Smart environment created with Wokwi. Real-time data from the sensors were sent to the ThingSpeak cloud to allow remote monitoring with graphics analysis. To analyze the sensor data collected from the environment, a Random Forest classification model was created in Google Colab to predict conditions that could be related to occupancy. The proposed AIoT system was successfully impressed by successfully performing smartness in LED control based on the occupancy of the area and sending the monitoring data to the cloud. Experimental results revealed successful real-time communication, accurate classification of the occupancy, and seamless integration of IoT, cloud computing and machine learning technologies.
