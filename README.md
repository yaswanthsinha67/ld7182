# ld7182
IOT-related artificial intelligence
This project presents an artificial intelligence-based smart energy monitoring system that uses ESP32 to construct cloud connectivity, environmental sensors, and machine learning algorithms. Temperature, humidity, light intensity, and motion detection were all tracked in a Wokwi-created simulated smart environment. The ThingSpeak cloud received real-time sensor data to enable remote monitoring with visual analysis. A Random Forest classification model was developed in Google Colab to evaluate the sensor data gathered from the surroundings and forecast circumstances that could be connected to occupancy.By effectively controlling LEDs intelligently depending on area occupancy and transmitting monitoring data to the cloud, the suggested AIoT system was impressive. The outcomes of the experiment demonstrated effective real-time communication, precise occupancy categorisation, and smooth integration of IoT, cloud computing, and machine learning technologies.
